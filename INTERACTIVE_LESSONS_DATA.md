# Données pour les Leçons Interactives

## Structure des Données

Chaque village contient 3 leçons :
1. **Vocabulaire** - QCM, Associations image-mot, Écoute audio
2. **Grammaire** - Exercices à trous, Sélection de particules
3. **Prononciation** - Enregistrement audio, Vérification

Après les 3 leçons : **Article Culturel** avec mini-quiz

---

## VILLAGE 1 : Welcome Village (환영 마을)

### Leçon 1 : Vocabulaire (10 points)
**Type : QCM + Association**

Questions :
1. QCM: "Quel est le mot coréen pour 'Bonjour'?"
   - Correct: 안녕하세요 (Annyeonghaseyo)
   - Options: 감사합니다, 미안합니다, 안녕히 가세요

2. ASSOCIATION: Image d'une maison + mot coréen 집 (jip - maison)
   - Options: 집, 학교, 병원, 은행

3. QCM: "Traduisez 'Merci' en coréen"
   - Correct: 감사합니다 (Gamsahamnida)
   - Options: 안녕하세요, 미안합니다, 괜찮습니다

### Leçon 2 : Grammaire (15 points)
**Type : Exercices à trous**

Questions :
1. FILL_BLANK: "안녕 ___ 세요" (Complétez avec la particule correcte)
   - Correct: 하
   - Explication: Particule de politesse

2. GRAMMAR: "Sélectionnez la bonne terminaison : 나는 학생 ___"
   - Correct: 입니다
   - Options: 이다, 입니다, 있습니다, 했습니다

3. FILL_BLANK: "저는 ___ 입니다" (Je suis ___)
   - Correct: 프랑스 사람
   - Explication: Nationalité

### Leçon 3 : Prononciation (10 points)
**Type : Enregistrement audio**

Mots à prononcer :
1. 안녕하세요 (Annyeonghaseyo)
2. 감사합니다 (Gamsahamnida)
3. 미안합니다 (Mianhamnida)

## VILLAGE 2 : Market Village (시장 마을)

### Leçon 1 : Vocabulaire (10 points)
**Thème : Marché et Commerce**

Questions :
1. QCM: "Quel est le mot pour 'marché'?"
   - Correct: 시장 (sijang)
   - Options: 가게, 상점, 백화점

2. ASSOCIATION: Image d'une pomme + 사과 (sagwa)
   - Options: 사과, 바나나, 딸기, 포도

3. QCM: "Comment dit-on 'Combien ça coûte?'"
   - Correct: 얼마예요? (Eolmayeyo?)
   - Options: 뭐예요?, 어디예요?, 누구예요?

### Leçon 2 : Grammaire (15 points)
**Thème : Nombres et Quantités**

Questions :
1. GRAMMAR: "Sélectionnez le nombre correct : 5 pommes"
   - Correct: 사과 다섯 개
   - Options: 사과 5개, 사과 오 개, 사과 다섯

2. FILL_BLANK: "저는 ___ 을/를 사고 싶어요" (Je veux acheter ___)
   - Correct: 사과
   - Explication: Particule d'objet direct

3. GRAMMAR: "Complétez : 이것은 ___ 입니다"
   - Correct: 비싸다
   - Options: 싸다, 비싸다, 싸다, 저렴하다

### Leçon 3 : Prononciation (10 points)
Mots à prononcer :
1. 시장 (sijang - marché)
2. 사과 (sagwa - pomme)
3. 얼마예요? (Eolmayeyo? - Combien?)

## Système de Points

- **Vocabulaire** : 10 points par leçon
- **Grammaire** : 15 points par leçon
- **Prononciation** : 10 points par leçon
- **Article Culturel** : 20 points
- **Quiz Culturel** : 20 points bonus

**Total par village** : 75 points
**Total pour 5 villages** : 375 points
